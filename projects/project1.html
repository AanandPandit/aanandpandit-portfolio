Vision-Based UAV for Search and Rescue Operations

Oct 2024 - Apr 2025
Final Year Project

Project Overview

This project focuses on developing an autonomous UAV (Unmanned Aerial Vehicle) system designed for search and rescue operations in disaster-stricken areas. Using advanced computer vision and AI technologies, the drone can autonomously detect and track human victims in real-time, significantly improving response times in critical situations.

Key Features

  • Real-time human detection and tracking using YOLOv8 object detection model
  • Autonomous navigation in disaster zones with obstacle avoidance
  • Live video streaming with AI-powered victim identification
  • GPS coordinate logging of detected victims for rescue team coordination
  • Thermal imaging integration for night operations and detecting victims under debris
  • Multi-drone coordination for large area coverage

Technical Implementation

The system employs YOLOv8, a state-of-the-art object detection algorithm, trained specifically on human detection datasets. The model runs on an onboard NVIDIA Jetson Nano for edge computing, enabling real-time processing without relying on cloud connectivity.

The drone's flight controller is based on ArduPilot, integrated with custom Python scripts for autonomous mission planning and execution. Computer vision processing is implemented using OpenCV and PyTorch frameworks.

Results & Impact

  • Achieved 80% increase in search efficiency compared to traditional manual search methods
  • Successfully demonstrated real-time victim detection with 92% accuracy in test scenarios
  • Reduced manual intervention by enabling autonomous operation in hazardous environments
  • Capable of covering 5x more area in the same timeframe compared to ground-based search teams

Technologies Used

Python YOLOv8 OpenCV PyTorch NVIDIA Jetson Nano ArduPilot Computer Vision Deep Learning Drone Technology

Challenges & Learning

One of the major challenges was optimizing the YOLOv8 model to run efficiently on the limited computational resources of the Jetson Nano while maintaining real-time performance. This required model quantization and optimization techniques.

Another significant challenge was ensuring stable communication between the ground station and the UAV in disaster scenarios where infrastructure might be damaged. This led to implementing a robust failsafe system with autonomous return-to-home capabilities.

Future Enhancements

  • Integration of multi-spectral cameras for enhanced detection capabilities
  • Implementation of swarm intelligence for coordinated multi-drone operations
  • Addition of emergency supply delivery mechanism
  • Development of mobile app for real-time mission monitoring and control
