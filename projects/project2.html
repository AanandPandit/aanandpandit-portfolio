IoT-Based Plant Monitoring System

Feb 2023 - Jul 2023
IoT Project

Project Overview

An intelligent IoT-based plant monitoring system designed to automate plant care by continuously monitoring soil moisture, temperature, and humidity levels. The system provides real-time data visualization through a mobile application and sends automated alerts when plants require attention, making plant care effortless and efficient.

Key Features

  • Real-time monitoring of soil moisture, temperature, and humidity
  • Automated watering alerts and notifications via Android app
  • Historical data tracking and visualization with charts
  • Multi-plant support with individual sensor modules
  • WiFi connectivity for remote monitoring from anywhere
  • Battery backup system for continuous operation
  • Customizable thresholds for different plant species

Technical Implementation

The system is built around the ESP32 microcontroller, which serves as the central processing unit and WiFi gateway. Multiple sensors including capacitive soil moisture sensors, DHT22 temperature and humidity sensors are connected to gather environmental data.

The ESP32 communicates with a Firebase Realtime Database to store sensor readings, enabling cloud-based data access. An Android application developed in Java retrieves this data and presents it in an intuitive dashboard with real-time graphs and alert notifications.

Results & Impact

  • Successfully reduced manual plant care and monitoring efforts by 80%
  • Tested across multiple plant species with 95% accuracy in moisture detection
  • Prevented over-watering and under-watering through timely alerts
  • Improved plant health and survival rate in test environments
  • Average response time of less than 2 seconds for sensor data updates

Technologies Used

ESP32 Arduino IDE C++ Android (Java) Firebase DHT22 Sensor Soil Moisture Sensor WiFi IoT

System Architecture

The system follows a three-tier architecture:

  • Sensor Layer: Physical sensors collect environmental data from plants
  • Processing Layer: ESP32 processes sensor data and manages WiFi communication
  • Application Layer: Android app provides user interface and data visualization

Data flows from sensors through the ESP32 to Firebase, where it's stored and made accessible to the mobile application. The app uses Firebase Cloud Messaging for push notifications when attention is needed.

Challenges & Learning

One significant challenge was calibrating the soil moisture sensors for different soil types and plant species. This required extensive testing and the development of calibration profiles for various scenarios.

Power management was another consideration, as the system needed to operate continuously while minimizing power consumption. This led to implementing deep sleep modes between sensor readings and optimizing WiFi connection protocols.
